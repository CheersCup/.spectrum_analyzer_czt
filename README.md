# .spectrum_analyzer_czt
本项目是以 CZT（Chirp Z-Transform）线性调频 Z 变换为核心算法的频谱分析仪，基于 xc7z020clg400-2 平台硬件实现（仅开发 PL 端），采用 Verilog/SystemVerilog 开发，配套完整的 Xilinx Vivado 工程文件。相较于传统 FFT 频谱分析，CZT 可对指定窄带频段实现高精度、高分辨率频域分析，完美解决了 FFT 在局部频域检测中资源开销大、分辨率不足的痛点。 项目包含完整的 CZT 核心算法模块、顶层控制逻辑、信号输入输出接口、仿真测试文件，代码采用模块化设计，注释规范、易于移植与二次开发，适用于 FPGA 数字信号处理教学、科研实验及便携式频谱分析仪器开发。
